# 褶宇宙 Fold-Space

> 摊开手掌生成时空薄片，握紧拳头将其揉皱或折叠进更深维度——一款基于计算机视觉的沉浸式记忆拓扑沙盘。

![gesture-demo](https://placeholder-for-foldspace-demo.gif)

## 快速开始

```bash
git clone https://github.com/yourteam/fold-space.git
cd fold-space
npm install
npm run dev -- --host
```

> 需要 HTTPS 以访问摄像头进行手势校准。打开 `https://localhost:5173` 并允许摄像头权限。

## 核心概念

时间不是线，而是可以被折叠、揉皱、重新展开的空间。
在**褶宇宙**中，每一个想法都是漂浮在虚空中的**时空薄片**（Chrono-Fold）：它有厚度、有温度、有质量。通过前置摄像头捕捉你的手势，你可以在三维拓扑中：

- **生成**：从掌心**展开**新的时空薄片（张开手掌）
- **关联**：**折叠**两个遥远的记忆，让它们重叠共鸣（捏合拖拽）
- **坍缩**：**揉皱**某个不再需要的时刻，看它坍缩成奇点（握拳挤压）
- **俯瞰**：**展开**时间轴，从宇宙视角俯瞰生命轨迹（双手缩放）

*所有视觉处理本地完成，你的宇宙只存在于你的设备中。*

## 技术栈

- **时空渲染引擎**: Three.js + React Three Fiber (WebGL 2.0)
- **物理褶皱模拟**: Cannon.js + 自定义 GLSL Shader
- **视觉识别层**: MediaPipe Hands (本地手势追踪)
- **时空数据结构**: 非欧几里得坐标系（X=时间轴，Y=情绪维度，Z=重要性深度）
- **本地存储**: IndexedDB + LZ-string
