# 褶宇宙 Fold-Space

&gt; **把你的生命时刻折叠进可触摸的时空褶皱里。**  
&gt; 摊开手掌生成时空薄片，握紧拳头将其揉皱或折叠进更深维度——一款基于计算机视觉的沉浸式记忆拓扑沙盘。

![gesture-demo](https://placeholder-for-foldspace-demo.gif)

## 核心概念

时间不是线，而是可以被折叠、揉皱、重新展开的空间。  
在**褶宇宙**中，每一个想法都是漂浮在虚空中的**时空薄片**（Chrono-Fold）：它有厚度、有温度、有质量。通过前置摄像头捕捉你的手势，你可以在三维拓扑中：

- **生成**：从掌心**展开**新的时空薄片（张开手掌）
- **关联**：**折叠**两个遥远的记忆，让它们重叠共鸣（捏合拖拽）
- **坍缩**：**揉皱**某个不再需要的时刻，看它坍缩成奇点（握拳挤压）
- **俯瞰**：**展开**时间轴，从宇宙视角俯瞰生命轨迹（双手缩放）

*所有视觉处理本地完成，你的宇宙只存在于你的设备中。*

## 为什么叫"褶宇宙"？

就像纸张的褶皱让二维平面拥有了三维的深度，**Fold-Space** 让你的线性时间拥有了可触摸的拓扑结构。那些被揉皱的焦虑、被折叠的灵感、被展平的幸福，都在这片虚空中形成了属于你的引力场。

## 手势交互协议（零点击操作）

| 手势 | 时空操作 | 视觉反馈 |
|------|---------|---------|
| 🖐️ **展开 (Expand)** | 手掌张开悬停 1.2 秒 | 从掌心**舒展**出发光薄片，伴随空间涟漪 |
| 🤏 **折叠 (Fold)** | 捏住薄片 A 拖向薄片 B | 两张薄片边缘**卷曲**融合，生成时空褶皱连线（爱因斯坦-罗森桥视觉）|
| ✊ **坍缩 (Collapse)** | 握拳包裹薄片并挤压 | 薄片**向内坍缩**（Gravitational Collapse），从三维揉皱成一维奇点后消散，伴随纸张纤维断裂的 ASMR 音效 |
| 👐 **拓扑变换** | 双手开合 | 相机视角**超光速飞行**（Z轴穿越），时间尺度从"此刻"拉伸到"永恒" |

## 技术栈

- **时空渲染引擎**: Three.js + React Three Fiber (WebGL 2.0)
- **物理褶皱模拟**: Cannon.js (薄片碰撞与折叠物理) + 自定义 GLSL Shader (纸张/空间扭曲效果)
- **视觉识别层**: MediaPipe Hands (本地手势追踪，21 关键点骨骼绑定)
- **时空数据结构**: 非欧几里得坐标系（X=时间轴，Y=情绪维度，Z=重要性深度）
- **本地存储**: IndexedDB (时空拓扑的持久化) + LZ-string (压缩褶皱数据)
