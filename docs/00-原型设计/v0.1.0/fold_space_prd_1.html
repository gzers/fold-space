# 褶宇宙 Fold-Space | 引导页 Landing 设计稿说明 ## 1. 页面整体构图 (Global Composition) - **构图方式**：中心对齐式 (Center-aligned Focus)，强调唯一的入口感。 - **视觉层级**： 1. 背景：深邃的星云动态背景，带有轻微的呼吸感和漂浮感。 2. 视觉中心：产品名称与核心价值主张。 3. 交互中心：强有力的“开始进入我的宇宙”主按钮 (CTA)。 4. 辅助层：底部隐私安全提示与侧边手势引导。 ## 2. 布局分区 (Layout Sections) - **Hero Section (60% Height)**: 位于屏幕上半部中心，包含 Logo、Slogan 及主视觉。 - **Action Zone (20% Height)**: 屏幕中下部，放置主 CTA 按钮及“跳过手势”的次级入口。 - **Information Footer (15% Height)**: 屏幕最底部，分布权限说明与隐私安全提示。 - **Gesture Hint (Floating)**: 屏幕两侧或按钮周围的轻量化气泡，展示核心手势示意图。 ## 3. 区块具体内容 (Section Content) - **Header**: - 褶宇宙 Fold-Space (标题) - 在时空薄片中，收藏你的私人宇宙 (副标题) - **Main Action**: - [开始进入我的宇宙] (高发光主按钮) - 或使用鼠标/触控进入 (次级文字链接) - **Privacy & Security**: - “仅在本地持久化存储，您的记忆不上传任何服务器。” - 状态标签：[摄像头已就绪] [本地存储已连接] - **Gesture Guide**: - 展开手掌：新建 - 捏合：拖拽 - 长握拳：回收至奇点 ## 4. UI 组件样式 (UI Component Styles) - **主按钮 (Space-Pulse Button)**: - 背景：#1a1b3a 半透明毛玻璃。 - 边框：1px 宽，渐变发光 (#ff6b35 到 #6b5b95)。 - 阴影：外发光 (Glow) 效果，模拟星体光晕。 - 悬浮态：光晕增强，文字轻微放大。 - **卡片 (Glass-Slice Card)**: - 背景：rgba(26, 27, 58, 0.4)，模糊度 20px。 - 圆角：16px。 - **图标 (Ethereal Icons)**: - 细线描边，带有微弱的内外发光。 ## 5. 页面文案 (Copywriting) - **Title**: 褶宇宙 - **Subtitle**: 浅 Fold-Space - **Slogan**: 每一片记忆，都是宇宙的褶皱。 - **CTA**: 开启我的宇宙之旅 - **Security Hint**: 私密空间：所有记忆片段仅保存在您的本地浏览器中，我们无法触及。 - **Gesture Info**: - ✋ 展开：唤醒新记忆 - 🤏 捏合：穿梭时空 - ✊ 握拳：归于奇点 ## 6. 动效建议 (Animation) - **Background**: 星云缓慢旋转与颗粒漂浮感 (Parallax effect)。 - **Entrance**: 页面加载时，标题从深处缓缓浮现，伴随轻微的模糊到清晰的转场。 - **Button**: 呼吸灯效果的发光边框。 - **Gesture Icon**: 循环播放的极简手势线条动画 (1.6s 周期)。 ## 7. 低保真线框结构图 (Lo-Fi Wireframe) ```text _______________________________________________________ | | | [ 褶宇宙 Fold-Space ] (Logo/Title) | | | | [ 每一片记忆，都是宇宙的褶皱 ] (Slogan) | | | | /-------------\ | | | (主视觉区) | | | \-------------/ | | | | [ 开启我的宇宙之旅 ] (Primary CTA) | | | | ( 鼠标/触控进入 ) (Secondary) | | | | [✋ 展开] [🤏 捏合] [✊ 握拳] (Gesture Guides) | | | | ___________________________________________________ | | [ 🔒 本地存储 ] [ 📷 摄像头 ] [ 您的记忆仅存在本地 ] | |_____________________________________________________| ```
