# 褶宇宙 Fold-Space | 主画布页 Main Canvas 设计稿说明 ## 1. 页面整体构图 (Global Composition) - **构图方式**：无限画布 (Infinite Canvas) + 悬浮浮窗 (Floating Overlays)。 - **视觉层级**： 1. 底层：深空背景 (Starfield/Nebula)，带有微弱的视差滚动。 2. 中层：记忆卡片 (Memory Slices) 与 关联线 (Link Lines)。 3. 顶层：悬浮状态栏、时间轴、搜索虫洞等交互组件。 ## 2. 布局分区与尺寸占比 (Layout Sections) - **Main Canvas (100% Full Screen)**: 主操作区，承载所有记忆卡片。 - **Top Status Bar (Height: 64px)**: 悬浮在顶部，包含品牌、视图切换、搜索入口。 - **Timeline Sidebar/Bottom (Width/Height: Fixed)**: 侧边或底部的时间轴，用于快速定位时间维度。 - **Singularity Trash (Bottom Right)**: 废纸篓奇点入口，常驻右下角。 - **Camera Preview (Top Right/Floating)**: 摄像头预览窗，展示手势识别状态。 ## 3. 卡片组件设计 (Memory Slice Card) - **视觉样式**： - 背景：`rgba(255, 255, 255, 0.05)` 毛玻璃，模糊度 12px。 - 边框：1px 细线，颜色根据情绪变化（例如：积极-橙，低落-紫）。 - 内容：记忆文本片段、缩略图、时间标签、关联数量。 - **状态设计**： - 悬浮态：边框发光增强，卡片轻微放大。 - 选中态：产生呼吸灯边缘效果。 - 连接态：卡片之间出现动态流光线条。 ## 4. 顶栏设计 (Top NavBar) - **左侧**：Logo「褶宇宙」+ 当前视图名称（时间流/星云/网络）。 - **中间**：搜索虫洞输入框（激活时扩展）。 - **右侧**：视图切换（Icon Group）、帮助、更多设置。 ## 5. 时间轴设计 (Timeline) - **形式**：极简线条，带有点状刻度。 - **交互**：点击或拖拽刻度，画布自动平滑滚动到对应时间的记忆簇。 ## 6. 废纸篓奇点设计 (Singularity Trash) - **视觉**：一个具有轻微引力动画效果的深紫色发光圆环。 - **规则**：卡片拖入后缩小消失，点击圆环可展开“废纸篓面板”进行恢复或彻底销毁。 ## 7. 页面状态设计 (Page States) - **空态 (Empty)**：中心显示“展开手掌，开始记录第一片记忆”。 - **搜索态 (Search)**：画布变暗，非匹配卡片透明度降至 10%，匹配卡片高亮。 - **待删除确认 (Delete Confirm)**：卡片悬浮在奇点上方，显示“释放以归于奇点”或“点击取消”。 ## 8. 页面文案 (Copywriting) - **视图切换**：[时间流] [星云] [网络] - **搜索占位符**：穿越记忆虫洞... - **状态提示**：正在感知手势... | 本地存储已加密 - **废纸篓**：归于奇点 (Singularity) ## 9. 动效建议 (Animation) - **卡片生成**：从中心向外扩散的波纹效果。 - **删除到奇点**：卡片被螺旋吸入，伴随 1.2s 的消散动画。 - **视图切换**：平滑的位移与缩放 (FLIP 动画)。 ## 10. 低保真线框结构图 (Lo-Fi Wireframe) ```text _______________________________________________________ | [褶宇宙] [搜索...] [流/云/网] [?] | [📷 预览] | |_____________________________________________________| | | | [ 卡片 A ] ------ (关联线) ------ [ 卡片 B ] | | | | | (时间流向) | | | | | [ 卡片 C ] (情绪：紫) | | | | | | [时间轴 | 2024 ... 2025] ( ( 奇点 ) ) | |_____________________________________________________| ```
