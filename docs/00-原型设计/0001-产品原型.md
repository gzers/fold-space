# 褶宇宙 Fold-Space 原型设计文档

**文档用途**：供设计师、前端开发者、AI 原型工具生成可交互原型
**版本**：v1.0
**画布基准**：1440×900（桌面端），支持响应式缩放

---

## 一、产品概述

**核心隐喻**：用户站在一片虚空中，周围漂浮着发光的「时空薄片」（日记卡片），像星云一样分布。每张薄片代表一个记忆节点，通过光线连接形成「记忆星座」。

**交互哲学**：
- **零点击**：所有核心操作通过手势完成，鼠标/触摸为降级方案
- **物理反馈**：每张薄片都有重量、弹性和破碎感
- **情绪可视化**：颜色=时间+情绪，形状=内容长度，光晕=关联密度

---

## 二、信息架构（页面地图）

```text
Landing（可选引导页）
    └── Main Canvas（主画布）── 唯一核心界面
            ├── 悬浮输入层（语音/标签录入）
            ├── 详情编辑舱（右侧滑出面板）
            ├── 废纸篓奇点（左下角黑洞）
            ├── 手势引导层（半透明教学 UI）
            └── 系统状态栏（顶部极简信息）
```

**单页面应用（SPA）**：除首次权限申请外，所有操作在主画布完成，无页面跳转。

## 三、核心界面详细设计

### 3.1 主画布（The Void）

**布局结构**：
```text
┌─────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────┤

│     │ 今天被老板...   │    位置：随机漂浮但有引力聚集          │
│     │ #工作 #焦虑     │    角度：轻微倾斜（±5°随机）           │
│     ╰─────────────────╯                                     │
│              │                                              │
│              ╰──────────✦────────→ ╭──────────╮            │
│                                    │ 🧘 冥想    │            │
│                                    │ 其实是我   │            │
│                                    │ 太敏感...  │            │
│                                    ╰──────────╯            │
│  [中央浮点] ← 相机焦点，手势操作锚点                        │

│  [← 过去 ] ●━━●━━●━━●━━●━━●━━●━━●━━●━━●━━●━━●━━● [ 现在→]    │  ← 时间褶皱轴（底部 60px）

│ 📷 摄像头   │  ← 右下角 120×90px 画中画，绿色状态点

```

**3D 场景参数**：
- **背景**：深空渐变 `#0a0a12` → `#1a1b3a`（径向渐变，中心较亮）
- **网格**：极淡的透视网格线（`rgba(255,255,255,0.03)`），仅在移动时可见，营造空间感
- **雾效**：Three.js FogExp2（`density: 0.02`），远处薄片渐隐，增强深度
- **光照**：环境光（`#404060`, intensity 0.5）+ 方向光（`#ffffff`, intensity 1.2，模拟月光）

### 3.2 时空薄片（Chrono-Fold Card）

**视觉规范**：

| 属性 | 规格 |
|------|------|
| **尺寸** | 宽度 280px，高度自适应（最小 120px，最大 400px，超出滚动） |
| **形状** | 圆角 16px，轻微不规则（顶点随机偏移 ±2px，模拟手工纸） |
| **材质** | 毛玻璃效果：`backdrop-filter: blur(20px)` + `background: rgba(255,255,255,0.08)` |
| **边框** | 1px solid `rgba(255,255,255,0.15)`，内发光 `box-shadow: 0 0 20px rgba(255,255,255,0.05)` |
| **厚度** | Three.js 中厚度 4px（ExtrudeGeometry），侧边颜色比正面深 20% |

**情绪色谱（正面）**：
- **深夜 0-4h**：`#1a1b3a`（深蓝紫）+ 冷白文字
- **清晨 5-8h**：`#ff6b35`（暖橙）+ 深棕文字
- **白昼 9-17h**：`rgba(255,255,255,0.9)`（透明白）+ 深灰文字
- **傍晚 18-23h**：`#6b5b95`（紫罗兰）+ 米白文字

**内容层级**：
```text
┌─────────────────────────┐
│ ○ 时间戳 · 情绪图标      │  ← 顶部 24px，12px 文字，透明度 0.6
│                         │
│   主要内容文本...       │  ← 16px 正文，行高 1.6，最大 6 行
│   第二行内容...         │
│                         │
│ #标签 #标签              │  ← 底部 32px，14px，金色高亮 #ffd700
└─────────────────────────┘
```

**状态变化**：
- **悬浮（Hover）**：放大 1.05 倍，阴影加深，边框变亮（`rgba(255,255,255,0.4)`）
- **选中（Active）**：周围出现 4 个发光锚点（可拖拽连接），轻微上下浮动动画暂停
- **锁定（Pre-Collapse）**：边缘发红光脉冲（`#ff3333`, 1Hz 闪烁），整体轻微震动（CSS shake 或 Three.js 抖动）

### 3.3 输入模式 UI（Voice Input）

**触发**：G-01 手势（展开手掌 1.2s）后，在手掌位置生成输入舱

**布局**：
```text
          ┌─────────────────────┐
          │   🎙️ 正在聆听...   │  ← 中央大图标，脉冲动画
          │                     │
          │   [波形可视化]      │  ← 音浪条，上下浮动
          │   ██████░░░░░░░░    │
          │                     │
          │   "今天被老板..."   │  ← 实时转写文字，逐字出现
          │                     │
          │  [✊ 握拳保存]       │  ← 底部提示，红色按钮样式
          └─────────────────────┘
                ↑
          从用户掌心位置向上浮出（Z轴动画）
```

**交互细节**：
- **录音中**：波形条根据音量高低跳动（Web Audio API 分析器）
- **倒计时**：刚开始时显示 3-2-1 淡入淡出，然后麦克风图标放大
- **手势保存**：检测到握拳（G-04 短触发）→ 波形条收束成一条直线 → 卡片「折叠」成实体（收缩动画）→ 飞向时间轴位置
- **取消**：手放下离开画面 3 秒，输入舱消散成粒子

---

### 3.4 坍缩动画（Collapse Animation）

**触发**：G-05 手势（握拳揉碎）

**四阶段动画**：

| 阶段 | 时长 | 视觉效果 | 音效 |
|------|------|----------|------|
| **预警** | 0-0.8s | 卡片边缘红色光晕增强，出现「裂纹」纹理（shader noise），整体轻微高频震动 | 低频嗡鸣（sub-bass），频率逐渐升高 |
| **起皱** | 0.8-1.2s | **顶点位移**：卡片几何体中心向拳心凹陷，边缘起皱（vertex displacement 向噪声），像被手捏皱的纸 | 纸张揉皱的「咯吱」声（ASMR） |
| **坍缩** | 1.2-1.5s | 卡片迅速缩小至 5%，向拳心吸入，同时透明度降至 0，发光强度增至最大（最后闪光） | 气球放气+玻璃碎裂的混合音 |
| **爆散** | 1.5-2.0s | **粒子爆发**：从拳心位置生成 150 个碎片（InstancedMesh），初始速度向内（模拟吸力），然后向外爆开，受重力下落 | 纸张撕裂的「刺啦」声+轻爆破声 |

**粒子规格**：
- 形状：不规则多边形（5-7 边），模拟碎纸片
- 大小：5-15px 随机
- 颜色：继承原卡片颜色，但饱和度 +30%
- 物理：Cannon.js 刚体，质量 0.1，有空气阻力（缓慢飘落）
- 生命周期：3 秒后渐隐消失（落入废纸篓奇点）

---

### 3.5 废纸篓奇点（Trash Singularity）

**位置**：左下角固定，圆形按钮 60×60px

**视觉**：
- **常态**：深邃黑洞效果（径向渐变 `#000000` → `#1a1b3a`），周围有微光吸积盘缓慢旋转
- **有内容时**：显示数字角标（红色），奇点周围有碎片漂浮（被捕获的卡片残骸）
- **Hover**：放大至 80px，吸积盘转速加快，产生「吸引力」视觉效果（周围薄片轻微向其偏移）

**交互**：
- **点击/捏合**：展开「奇点视界」面板（从黑洞向上滑出），显示最近坍缩的 5 张卡片（模糊预览）
- **恢复**：拖拽卡片出黑洞 → 播放「白洞蒸发」反向动画（碎片重组 → 展开成完整卡片）
- **彻底粉碎**：在奇点视界内长按卡片 2s → 卡片被吸入中心消失（不可逆）

### 3.6 手势引导层（Ghost Layer）

**首次使用或检测到新手势错误时显示**

**幽灵手（Ghost Hand）**：
- 半透明绿色轮廓（`rgba(100,255,150,0.3)`），骨骼线框风格
- 在屏幕中央演示正确手势（展开/捏合/握拳）
- 用户模仿成功后，幽灵手消散成粒子（正向反馈）

**提示文字**：
```text
┌─────────────────────────────┐
│  👆 请张开手掌              │
│  保持 1 秒生成新记忆        │
│                             │
│     [幽灵手动画演示]        │
│         🖐️ → ✋            │
└─────────────────────────────┘
```

**动态提示（运行时）**：
- 底部 40px 高半透明条，显示当前检测到的手势状态：
  - 检测到手：「🖐️ 检测到手部」
  - 可展开：「⏳ 保持展开... 3,2,1」
  - 可坍缩：「✊ 用力握拳揉碎」

## 四、组件库（Component Library）

### 4.1 基础组件

**C1. 发光连线（Fold-Link）**
- 连接两张薄片
- 贝塞尔曲线，中段向外弯曲（像时空弯曲）
- 材质：发光管线（`LineBasicMaterial`，颜色根据关系类型变化）
  - 因果关系：`#00ffff`（青）
  - 灵感关系：`#ffd700`（金）
  - 对立关系：`#ff0055`（红）
- 动画：粒子在连线上缓慢流动（`texture offset`）

**C2. 时间褶皱轴（Time-Axis）**
- 底部水平条，非线性刻度（记录密集处刻度压缩，稀疏处拉伸）
- 可拖拽（改变相机 Z 轴位置）
- 当前视角位置用发光三角标记 ▲

**C3. 粒子系统（Particles）**
- 环境粒子：漂浮的尘埃（模拟宇宙尘埃），对相机移动有视差反应
- 交互粒子：手势轨迹（手移动时留下 0.5 秒消散的光迹）

### 4.2 状态组件

**S1. 加载态（Singularity Boot）**
- 中央旋转的黑洞吸积盘动画
- 文字：「正在展开时空拓扑...」

**S2. 空态（Empty Void）**
- 中央悬浮提示：「🖐️ 张开手掌，从这里开始你的宇宙」
- 背景有示例薄片幽灵（半透明，示意最终效果）

## 五、交互规范（Interaction Spec）

### 5.1 手势反馈映射

| 用户动作 | 界面反馈 | 延迟要求 |
|----------|----------|----------|
| 手进入画面 | 右下角摄像头预览窗绿色边框闪烁 1 次 | < 100ms |
| 张开手掌 | 掌心位置出现 3 秒倒计时环（逐渐填满的圆环） | 实时 |
| 手掌保持 1.2s | 展开涟漪特效（Shockwave Shader），输入舱浮出 | 0ms（预加载）|
| 捏合检测 | 目标薄片边缘立即形变（< 50ms），光标变为「捏住」图标 | < 50ms |
| 握拳检测 | 0.3s 内薄片变红 + 震动，0.8s 后进入锁定态 | 实时 |
| 揉碎执行 | 立即触发坍缩四阶段动画，不可中断 | 0ms |
| 手离开画面 | 3s 后暂停手势识别，显示「👋 请举手」提示 | 3s 缓冲 |

### 5.2 错误处理 UI

| 错误类型 | 反馈方式 |
|----------|----------|
| 手势置信度低 | 屏幕边缘黄色脉冲 + 文字「请保持手部在画面中」 |
| 误触（突然握拳） | 薄片轻微抖动但不变红（阈值过滤） |
| 存储失败 | 顶部红色横幅「无法访问本地文件，已降级为浏览器存储」 |
| 麦克风拒绝 | 输入舱切换为键盘图标，提示「请手动输入或允许麦克风」 |

## 六、视觉规范（Design System）

### 6.1 颜色系统

```css
/* 深空背景 */
--void-bg: #0a0a12;
--void-grid: rgba(255,255,255,0.03);

/* 卡片基础 */
--fold-glass: rgba(255,255,255,0.08);
--fold-border: rgba(255,255,255,0.15);
--fold-glow: rgba(255,255,255,0.1);

/* 情绪色谱 */
--emotion-night: #1a1b3a;    /* 深夜 */
--emotion-dawn: #ff6b35;      /* 清晨 */
--emotion-day: #f0f0f0;       /* 白昼 */
--emotion-dusk: #6b5b95;      /* 傍晚 */
--emotion-anxiety: #ff3333;   /* 焦虑红 */
--emotion-calm: #00ffff;      /* 平静青 */

/* 功能色 */
--alert-lock: #ff3333;        /* 锁定/删除 */
--success-restore: #00ff88;   /* 恢复 */
--ghost-hand: rgba(100,255,150,0.3);
```

### 6.2 字体规范

- **主字体**：`Inter` 或 `PingFang SC`（系统默认无衬线）
- **代码/数据**：`JetBrains Mono`（时间戳、技术信息）
- **标题**：`Space Grotesk`（未来感，用于 Logo 和标题）

**层级**：
- 时间戳：12px，字重 400，透明度 0.6
- 正文：16px，字重 400，行高 1.6
- 标签：14px，字重 600，颜色 `--emotion-dawn`
- 提示文字：14px，字重 500，发光效果（`text-shadow`）

### 6.3 动效参数

**缓动函数**：
- 漂浮：自定义正弦波 `sin(t * 0.5) * 5px`，每张薄片相位不同（自然感）
- 交互：`cubic-bezier(0.34, 1.56, 0.64, 1)`（弹性过冲）
- 坍缩：`cubic-bezier(0.7, 0, 0.84, 0)`（加速吸入）

**时长**：
- 微交互（Hover）：200ms
- 状态切换（展开/折叠）：400ms
- 复杂动画（坍缩）：2000ms（四阶段总和）
- 环境动画（漂浮）：无限循环，周期 4-6s 随机

## 七、响应式适配

### 7.1 桌面端（Primary）
- 分辨率：1440×900 及以上
- 交互：手势（主）+ 鼠标（辅）
- 布局：完整三栏（画布中央，编辑舱右侧，时间轴底部）

### 7.2 平板端（iPad Pro 等）
- 分辨率：1024×768 及以上
- 适配：编辑舱改为底部半屏弹出（避免遮挡 3D 场景）
- 性能：粒子数量减半（75 个），简化物理计算

### 7.3 移动端（降级模式）
- **手势不可用**（摄像头通常朝后，或持握不稳）
- **纯触控模式**：
  - 长按空白处 = 新建
  - 双指缩放 = 时间轴缩放
  - 摇一摇 = 撤销（利用陀螺仪）
- **视觉**：简化 3D 为 2.5D（CSS 3D transform），确保 60fps
