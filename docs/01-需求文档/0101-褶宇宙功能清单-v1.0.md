# 褶宇宙 Fold-Space 功能清单 v1.1

> **把你的生命时刻折叠进可触摸的时空褶皱里。**
> 首版目标不是一次性实现所有想象，而是验证“新建、浏览、连接、删除恢复”这四个核心动作是否成立。

---

## 一、核心概念

- **时空薄片 (Chrono-Fold)**：每张日记卡片是空间中的记忆节点，具有时间、情绪、标签与关联
- **拓扑操作**：通过轻量手势或鼠标完成创建、拖拽、连接与回收
- **零点击优先**：核心路径可用手势完成，但所有能力必须有显式兜底入口
- **绝对本地优先**：视频流、文本和结构化数据均保留在本地

## 三、功能模块清单

### 3.1 时空画布 (Fold-Canvas)
- [ ] **主画布容器**：沉浸式空间背景与卡片排布
- [ ] **时间流视图**：默认视图，时间是主轴
- [ ] **标签星云视图**：按标签形成聚类
- [ ] **关联网络视图**：突出节点和连线
- [ ] **基础渲染性能**：首版稳定支持 100 张卡片、30 条高亮连线

### 3.2 记忆管理 (Chrono-Fold)
- [ ] **薄片 CRUD**：
  - 创建：手势展开或点击新建
  - 读取：点击卡片或选中展开详情
  - 更新：原位轻编辑 + 编辑舱深编辑
  - 删除：先进入废纸篓，再决定是否永久删除
- [ ] **标签提取**：支持 `#标签`
- [ ] **关联建立**：拖拽卡片到目标卡片上建立关系
- [ ] **本地持久化**：IndexedDB 存储

### 3.3 日记录入系统 (Chrono-Input)
- [ ] **语音速记**：作为推荐输入方式，但需允许失败兜底
- [ ] **极简录入**：只选情绪和标签，也可先创建空白卡片
- [ ] **键盘输入**：在无法录音或用户拒绝权限时立即可用
- [ ] **静默确认态**：转写后自动补全标签建议与情绪建议

### 3.4 辅助功能
- [ ] **幽灵引导系统**：按动作逐个教学
- [ ] **搜索虫洞**：关键词高亮 + 视角飞行定位
- [ ] **废纸篓奇点**：删除回收、恢复、永久清除
- [ ] **轻量模式**：低性能或低光环境下自动降级
- [ ] **私密模式**：模糊内容，仅显示情绪色块和基础信息

## 四、首版手势操作清单

**环境前提**：前置摄像头启用，手部距离 0.5m-1.5m，光照充足，HTTPS 环境

### 4.1 必做手势

| 手势 ID | 中文名 | 触发条件 | 主功能 | 优先级 |
|---------|--------|----------|--------|--------|
| **G-01** | 展开 | 五指张开并保持 1.2s | 新建薄片 | P0 |
| **G-02** | 捏合 | 拇指与食指靠近 | 拖拽薄片 | P0 |
| **G-03** | 释放连接 | 拖拽到另一卡片上释放 | 建立关联 | P0 |
| **G-04** | 长握拳 | 靠近卡片并保持 | 删除到废纸篓 | P1 |

### 4.2 可选补充手势

| 手势 ID | 中文名 | 主功能 | 优先级 |
|---------|--------|--------|--------|
| **G-05** | 双手开合 | 缩放视图 | P2 |
| **G-06** | 左挥撤销 | 撤销上一步 | P3 |

## 五、降级与辅助操作

当摄像头不可用、光照不足或手势识别置信度过低时，自动切换为触控/鼠标模式：

| 原手势 | 鼠标/触控替代 | 说明 |
|--------|--------------|------|
| 展开新建 | 点击 `+` / 右键新建 / 长按空白处 | 必须始终可见 |
| 捏合拖拽 | 鼠标拖拽 / 单指拖拽 | 与手势语义保持一致 |
| 建立连接 | 拖到目标卡片上释放 | 悬停时高亮目标 |
| 删除到废纸篓 | 长按卡片 / 右键“移入废纸篓” | 不允许直接不可逆删除 |
| 缩放视图 | 滚轮 / 双指缩放 | 轻量实现即可 |

## 六、手势容错与边界处理

| 边界情况 | 处理方案 | 用户提示 |
|----------|----------|----------|
| 手离开画面 | 暂停识别，保持当前状态 | 「请把手移回画面中央」 |
| 误触握拳 | 未靠近卡片时不触发删除 | 小图标提示但不执行动作 |
| 删除中途取消 | 手势解除即回退原状态 | 卡片恢复原位 |
| 低光/背光 | 自动切鼠标模式 | 「光线不足，已切换轻量交互」 |
| 性能不足 | 减少粒子、阴影和连线动画 | 「已切换轻量模式」 |
| 多人入镜 | 仅识别主手 | 非主手灰显 |

---

## 七、隐私与安全

- **零上传**：摄像头视频流仅本地处理
- **零服务器**：首版不依赖后端
- **本地存储**：优先 IndexedDB
- **加密导出**：作为后续迭代，不阻塞首版原型
- **录音不保存**：转写后立即丢弃原音频流

---

## 八、技术实现检查点

- [ ] **MediaPipe Hands** 或等效方案接入
- [ ] **手势状态机**：空闲 → 检测 → 执行 → 冷却
- [ ] **2D 到 3D 映射**：保证拖拽与目标命中可靠
- [ ] **基础卡片动画**：选中、拖拽、连接、回收
- [ ] **轻量删除动画**：伪物理即可，不强依赖刚体
- [ ] **搜索定位**：结果高亮与相机移动
- [ ] **本地存储**：至少保证文本、标签、位置恢复
- [ ] **轻量模式**：可手动或自动开启
