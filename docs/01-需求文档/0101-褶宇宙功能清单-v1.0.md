# 褶宇宙 Fold-Space 功能清单 v1.0

> **把你的生命时刻折叠进可触摸的时空褶皱里。**
> 摊开手掌生成时空薄片，握紧拳头将其揉皱或折叠进更深维度——一款基于计算机视觉的沉浸式记忆拓扑沙盘。

---

## 一、核心概念

- **时空薄片 (Chrono-Fold)**：每张日记卡片是三维空间中的实体，具有厚度、情绪色温、质量与空间坐标
- **拓扑操作**：通过手势对时间进行非线性管理（折叠关联、坍缩删除、展开创建）
- **零点击交互**：全程基于摄像头手势识别，无需触碰鼠标/键盘即可完成核心流程
- **绝对本地优先**：所有数据与视频流仅存在于本地设备，不上传任何服务器

---

## 二、功能模块清单

### 2.1 时空画布 (Fold-Canvas)
- [ ] **无限拓扑空间**：非欧几里得坐标系（X=时间轴，Y=情绪维度，Z=重要性深度）
- [ ] **薄片渲染引擎**：基于 Three.js InstancedMesh，单场景支持 1000+ 张记忆卡片
- [ ] **情绪光谱着色**：6时段自动配色（深夜蓝/清晨橙/白昼白/黄昏紫）+ 手动色温调节
- [ ] **引力场视觉**：相关薄片靠近时产生轻微吸引（视觉暗示）
- [ ] **时空褶皱连线**：贝塞尔曲线连接关联记忆，支持 3 种关系类型（因果/灵感/对立）
- [ ] **星云俯瞰模式**：双手开合缩放，从秒级微观到年级宏观的相机飞行

### 2.2 记忆管理 (Chrono-Fold)
- [ ] **薄片 CRUD**：
  - 创建：手势展开 + 语音/标签输入
  - 读取：点击/捏合展开详情面板
  - 更新：原位编辑或关联新连接
  - 删除：握拳坍缩动画（可恢复）
- [ ] **标签星云**：自动提取 `#标签`，同标签薄片在俯瞰视图形成聚类光晕
- [ ] **废纸篓奇点**：坍缩的薄片进入可恢复状态（压缩成奇点），支持「白洞蒸发」式恢复动画
- [ ] **本地持久化**：IndexedDB 存储，支持导出 `.fold` 加密文件（LZ-string + AES-256）

### 2.3 日记录入系统 (Chrono-Input)
- [ ] **语音速记 (默认)**：Web Speech API 本地识别，手势控制录音启停
- [ ] **标签速记 (极简)**：仅选择情绪色温 + 标签，无需语音或打字
- [ ] **实体键盘 (Fallback)**：双击展开编辑舱，支持 Markdown 语法与 AI 辅助
- [ ] **智能结构化**：自动提取标签、情绪关键词、时间戳并映射到空间坐标

### 2.4 辅助功能
- [ ] **幽灵引导系统**：首次使用的强制手势校准（跟随半透明示范手）
- [ ] **时空轴导航**：底部非线性时间滑块（密度反映记录频率）
- [ ] **搜索虫洞**：关键词高亮 + 视角飞行定位
- [ ] **私密模式**：一键模糊所有内容，仅显示情绪色块
- [ ] **错误恢复**：挥手撤销（Undo）上一次操作

## 三、手势操作清单（核心）

**环境前提**：前置摄像头启用，手部距离 0.5m-1.5m，光照 > 200lux，HTTPS 环境

### 3.1 基础手势集

| 手势 ID | 中文名 | 英文名 | 判定逻辑 | 触发条件 | 主功能 | 视觉/触觉反馈 |
|---------|--------|--------|----------|----------|--------|---------------|
| **G-01** | 展开 | **Expand** | 五指伸直分开，掌心朝向摄像头，指尖距掌心 > 80% 臂长 | 保持 **1.2s** | **新建薄片** | 掌心发光 → 空间涟漪 → 薄片从手心「生长」展开，伴随纸张展开 ASMR 音效 |
| **G-02** | 捏合 | **Pinch** | 拇指与食指指尖距离 < 30px（画面坐标），其余三指自然弯曲 | 瞬时触发 | **拖拽薄片** | 捏住时薄片边缘产生物理形变（UV 扭曲），跟随手指延迟 50ms（有质量感） |
| **G-03** | 折叠 | **Fold** | 在 **Pinch** 状态下，将薄片 A 拖至薄片 B 中心 50px 范围内释放 | 释放触发 | **建立关联** | 两张薄片边缘卷曲融合 → 生成发光「虫洞连线」（爱因斯坦-罗森桥视觉），伴随水晶碰撞音效 |
| **G-04** | 握拳 | **Fist** | 五指指尖卷曲向掌心，指尖距腕点 < 60% 掌长，拇指内扣压食指 | 保持 **0.8s** | **锁定薄片** | 目标薄片边缘出现红色警示光晕 + 轻微震动（Vibration API 200ms），提示「进入坍缩预备」 |
| **G-05** | 坍缩 | **Collapse** | 在 **Fist** 锁定状态下，手腕旋转速度 > 45°/s 或握拳加深 | 瞬时触发 | **揉碎删除** | 薄片向内坍缩（Gravitational Collapse）：<br>1. 起皱（顶点位移噪声）<br>2. 吸入（向拳心缩放）<br>3. 爆散（150 粒子从指缝喷出）<br>音效：揉纸团咯吱声 + 爆破噗声 |
| **G-06** | 拓扑开合 | **Scale** | 双手同时出现，双手掌心距离变化（张开/合拢） | 持续变化 | **时间轴缩放** | 双手拉开 = 超光速后退（相机 Z 轴后退显示全景）；合拢 = 进入微观（聚焦单一时段） |
| **G-07** | 挥手撤销 | **Undo** | 单手快速向左挥动（水平速度 > 300px/s，持续 < 0.5s） | 瞬时触发 | **撤销上一步** | 时间倒流特效（画面轻微反色 + 粒子倒飞），操作反向动画 |
| **G-08** | 托举平移 | **Pan** | 单手平摊，水平移动手掌（掌心朝下，移动速度 < 200px/s） | 持续触发 | **画布平移** | 视角跟随手方向平滑漂移，像推动一片星云 |

### 3.2 组合手势（高阶）

| 组合 ID | 前置手势 | 衔接动作 | 功能 | 特效 |
|---------|----------|----------|------|------|
| **C-01** | 捏合 (G-02) → 快速握拳 | 捏住薄片瞬间握拳（<0.3s） | **紧急坍缩** | 跳过 0.8s 锁定，直接触发揉碎（瞬间宣泄） |
| **C-02** | 展开 (G-01) + 双手拓扑 (G-06) | 新建时双手拉开 | **创建并俯瞰** | 新生成的薄片自动缩小并归入时间轴全景 |
| **C-03** | 双击捏合 | 快速两次捏合同一薄片 | **展开详情** | 薄片翻转 180°（像翻扑克牌），背面显示完整内容与关联图谱 |

### 3.3 录入控制手势（语音输入时）

| 手势 ID | 动作 | 功能 | 判定 |
|---------|------|------|------|
| **I-01** | 短握拳 (0.3s) | **停止录音并保存** | 语音输入模式下，G-04 短触发（避免与坍缩混淆） |
| **I-02** | 双手捂嘴 | **暂停/静音** | 双手靠近嘴部 landmarks（隐私保护，防止误录） |
| **I-03** | 单手左右挥动 | **切换标签** | 标签选择模式下，水平挥手切换类别 |
| **I-04** | 单手上下挥动 | **调节情绪** | 标签选择模式下，垂直挥手调节色温（上=暖/积极，下=冷/消极） |

## 四、降级与辅助操作（无摄像头/低光环境）

当摄像头不可用或手势识别置信度 < 60% 持续 5 秒时，自动切换为「触控/鼠标模式」：

| 原手势 | 鼠标/触控替代 | 视觉保持 | 备注 |
|--------|--------------|----------|------|
| 展开 (G-01) | 右键点击空白处 / 长按空白 1s | 同样出现「从点击处生长」动画 | 右键菜单第一项「新建时空薄片」 |
| 捏合拖拽 (G-02) | 左键按住拖拽 / 单指按住滑动 | 薄片边缘依然有轻微形变（CSS transform） | 拖拽时显示「虚影连线」提示可连接目标 |
| 折叠 (G-03) | 拖拽到目标上释放 / 点击「连接按钮」 | 同样的虫洞连线特效 | 鼠标悬停目标时，目标高亮提示 |
| 握拳坍缩 (G-05) | 右键点击薄片 → 选择「坍缩」/ 长按薄片 2s | **保留核心特效**：长按 2s 后自动播放揉碎动画（模拟手势锁定+坍缩过程） | 必须有揉碎动画，这是产品核心体验 |
| 拓扑开合 (G-06) | 滚轮缩放 / 双指捏合缩放 | 相机 Z 轴移动效果保持一致 | 滚轮阻尼手感模拟「空间厚重感」 |
| 挥手撤销 (G-07) | Ctrl+Z / 两指向左滑动 | 同样的时间倒流粒子特效 | 键盘快捷键保持 |

## 五、手势容错与边界处理

| 边界情况 | 处理方案 | 用户提示 |
|----------|----------|----------|
| **手离开画面** | 暂停手势识别，保持当前画布状态 10s 后自动切换鼠标模式 | 屏幕边缘显示「👋 请举起手」淡入提示 |
| **双手冲突**（一手想展开一手想握拳） | 优先识别距离摄像头更近的手（画面占比大者），另一只手忽略 | 被忽略的手显示为半透明灰色，主手高亮绿色 |
| **误触握拳**（紧张时自然握拳） | 必须满足「握拳 + 靠近薄片 (< 150px 画面距离)」才触发锁定，单纯握拳不操作 | 单纯握拳时显示「✊ detected」小图标但无红色警示 |
| **快速连续坍缩** | 每次坍缩后 0.5s 「手势冷却期」，防止误删 | 冷却期内屏幕边缘显示淡红色进度条 |
| **低光/背光** | 置信度 < 60% 时自动切换鼠标模式，同时弹出提示 | 「光线不足，已切换触控模式」 |
| **多人在画面** | 仅识别最大的人手（或中央区域的手），忽略其他 | 其他人的手显示为灰色剪影，主手绿色骨骼 |

## 六、隐私与安全

- **零上传**：摄像头视频流经由 MediaPipe 在浏览器本地实时处理，**永不触碰网络**
- **零服务器**：无后端 API，无用户数据库，无数据分析
- **本地加密**：数据存储使用 AES-256 (用户可选密码)，设备丢失也无法被读取
- **录音不保存**：语音转写完成后，音频流立即丢弃，仅保留文本
- **私密模式**：说"这是秘密"或手势 **双手捂住嘴(I-02)** 启动，此条日记显示为黑色薄片，需特定手势序列解锁

## 七、技术实现检查点

- [ ] **MediaPipe Hands** 集成：获取 21 关键点，手部边界框估算距离
- [ ] **手势状态机**：空闲 → 检测手 → 识别手势 → 执行操作 → 冷却 → 空闲
- [ ] **射线检测**：将 2D 手势坐标映射到 Three.js 3D 空间，准确「捏住」虚空中的薄片
- [ ] **物理形变 Shader**：薄片被捏时实时顶点位移（GLSL noise + 手指位置 uniform）
- [ ] **粒子坍缩系统**：InstancedMesh 粒子，初始速度向心（吸入）→ 然后爆炸（爆散）→ 重力下落
- [ ] **Vibration API**：在支持的设备触发触觉反馈
- [ ] **Web Speech API**：本地语音识别，流式转写
- [ ] **强制手势教学**：首次使用必须完成「展开→捏合→坍缩」三连才能进入主界面
